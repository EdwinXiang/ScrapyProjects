　　虽然单反相机代表着专业，但是把单反相机买回来后就让其在角落里吃灰的也不在少数，尤其是非专业用户，一大原因就是因为单反携带不够方便，所以现在大家更倾向于选择便携又不失专业的大底便携类相机，不仅同样能拍摄出高画质的照片，并且还能实现轻松出行。而此类机型中佳能PowerShot G系列一直凭借着娇小身段和专业操控著称，深受摄影玩家喜爱，也是不少人备机的首选。就在不久前佳能最新推出的PowerShot G1 X Mark III更是直接搭载了更大的APS-C画幅，并且还是全球首款搭载变焦镜头的APS-C画幅CMOS固定镜头便携相机，性能强大之余更是满足了不少用户对于便携性与高画质并存的需求。　　性能方面，佳能G1 X Mark III搭载2420万像素APS-C画幅CMOS，同时首次应用了全像素双核CMOS AF技术，并配备DIGIC 7影像处理器，最高常用感光度达到ISO 25600。其实早在2012年佳能就曾推出过大尺寸CMOS的G1 X数码相机，在当时还是1/2.3英寸传感器满天飞的DC市场，佳能G1 X就以其1.5英寸CMOS吸引了不少人目光，此后其他更低阶产品如G3 X、G5 X、G7 X和G9 X也都配备了1英寸CMOS。但这次佳能PowerShot G1 X Mark III横空出世，作为主打专业市场的大底相机不仅延续了佳能Powershot G系列轻巧便携的特色，更是致力于为大家带来更专业操控与更好的画质及性能。　　佳能G1 X Mark III搭载了与单反一样的APS-C画幅CMOS感光元件　　　　相比于上代方正造型的G1 X Mark III来说，这次全新的G1 X Mark III的外形设计有很大变化，一方面是增加了EVF电子取景器，并且佳能还给EVF设计了一个凸起的军舰部，整体看上去更具专业范；另一方面，无论是造型设计还是各个转盘以及按键的操控布局也与上代很不一样，看上去更接近此前推出的G5 X。虽然搭载了APS-C画幅CMOS感光元件，但G1 X Mark III的体型控制非常优秀，115.0×77.9×51.4mm的机身尺寸与399g（含电池和储存卡）的重量，仅比1英寸CMOS的G5 X稍稍大一圈（G5 X：112.4×76.4×44.2mm，377g），要知道APS-C画幅传感器的大小比1英寸大了不止两倍。　　佳能G1 X Mark III外观　　佳能G1 X Mark III外形设计更接近G5 X，而非上代产品（左：G1 X Mark II / 右：G5 X ）　　虽然G1 X Mark III造型与G5 X很相似，但给我的感觉G1 X Mark III线条更加好看，尤其是“溜肩”的设计与整体风格更搭，不像G5X那样军舰部显得十分突兀。　　G1 X Mark III机顶按键布局与G5 X一样，左边为拍摄模式转盘，右侧为电源、快门/变焦拨杆以及曝光补偿转盘。另外，还配备标准热靴，可外接更大输出功率的闪光灯等扩展配件。　　内置闪光灯，但并不是弹起式，使用时需要手动抬起闪光灯　　此次佳能这款G1 X MarK III最大的特色应该就是APS-C画幅与变焦镜头的搭配组合了，以往像富士X100系列、理光GR、尼康Coolpix A，以及徕卡X-E等虽然也都是便携式APS-C画幅卡片机，但它们无一例外都搭载的是定焦头，而G1 X MarK III应该是目前市面上首款搭载变焦镜头的APS-C画幅便携式固定镜头相机。　　搭载24-72mm F2.8-5.6镜头（等效35mm）　　镜头规格方面， G1 X Mark III采用了为大尺寸图像感应器而设计的3倍光学变焦镜头，镜头等效焦距为24-72mm。该镜头广角端具备f/2.8大光圈，配合APS-C画幅以及9叶片光圈，能够拍摄出具有漂亮虚化效果的人物或静物照片。虽然相比上代的5X光学变焦有减少，但是毕竟是APS-C画幅，而24-72mm的黄金焦段也算是非常实用了，不少APS-C单反搭配的套机头规格和素质还远不如它。　　多功能控制环，可自定义设置多项参数调整，还可进行微调对焦，让拍摄更方便　　相机在关机-开机-变焦情况下，镜头筒长度变化　　机背所有的操控按键就位于右侧　　佳能G1 X Mark III的触摸屏可180度侧翻，无论是仰视还是俯视拍摄，都可以轻松实现，对于视频拍摄也非常有帮助。　　拍摄模式转盘　　佳能G1 X Mark III型号标识，转盘四周的菱形切割工艺在使用时触感非常好，并且每个转盘甚至快门按钮下方都设计有佳能Powershot G系列的小红圈。　　数据与音频接口，支持USB充电，侧面下方则是WiFi连接按键　　采用与其他G系列卡片机一样的1250毫安电池　　佳能G1 X Mark III与佳能EOS 200D外观对比　　佳能G1 X Mark III与富士X100外观对比　　同为APS-C画幅机型，佳能EOS 200是目前世界上最小巧的单反相机，可以轻松托于掌间，但G1 X Mark III与之相比仍显得更加袖珍，同时厚度更有优势，电动变焦镜头让G1 X Mark III完全可以亲身装进上衣口袋。而与定焦镜头的富士X100相比，G1 X Mark III也要小那么一圈。另外还值得一提的是，除了配备EVF电子取景器以及专业操控外，G1 X Mark III还具备防尘防水滴性能，相机的电池仓开闭处、各种操作按钮周围和镜筒使用了密封材料，在转盘旋转轴等没有使用密封材料的部位，零件制作精度较高，以减小零件之间的空隙，减少灰尘和水滴意外进入相机内部，更适合户外极端环境下安心拍摄。这是不少同类型相机，包括佳能Powershot G系列其他所不具备的，这足以证明其APS-C画幅旗舰DC的地位。
