　　作者 | 刘玭　　9月12日，USNews发布了全新的2018美国最佳大学排名。　　作为排名界最有权威和影响力的榜单，单看排名，今年与去年一样，普林斯顿夺得榜首，哈佛大学排第二，芝加哥大学和耶鲁大学位列第三。　　除了发挥稳定的选手，以下几所学校的学生已经炸了：　　我们学校才排XX名？不服！　　我们学校哪儿比不上XX，排名却差这么多，不服！　　　　　　2018　　2017　　有同学已经拍案而起：这位选手的名次比去年降了两名，就因为没钱吗？　　这所学校是波士顿五大名校（哈佛、麻省理工、塔夫茨、波士顿学院，布兰迪斯大学）之一，也是25所“新常春藤”成员之一。　　从学生录取、教师的教学到学术研究、教学和科研设施的建立，以及对教师的业绩评估等，它都遵从以质取胜、注重革新的办学原则。　　它拥有美国最古老和最富盛名的国际关系研究生院之一：弗莱彻法律与外交学院；而塔夫茨医学中心 ，是著名的生物医学研究中心，也是塔夫茨大学医学院的主要教学医院；Friedman School，营养科学与政策学院在营养学上名声在外，是学术界的标杆之一。　　　　　　2018　　2017　　这位选手下降了8名。　　有学生疑问：UIUC招谁惹谁了？也有人说，从形象上来说，它一点都不冤。　　这所享有世界声望的顶尖研究型大学，也是美国“十大联盟”的创始成员，美国大学协会成员，被誉为“公立常春藤”，与加州伯克利、密歇根安娜堡并称“美国公立大学三巨头”。　　其工程学院在全美乃至世界堪称至尊级地位，位于美国大学工程院排名前5，几乎所有工程专业均在全美排名前10，电气、土木、材料、农业、环境、机械等专排全美前5，计算机专排全美第2；商学院会计专排全美前2。　　该校拥有全美第三大的大学图书馆，仅次于哈佛、耶鲁。　　　　　　2018　　2017　　这位选手的名次是大家争议最大的——今年又又又又降了！　　它在2017US News世界大学排名第11，2018泰晤士世界大学排名第25名；2017CWUR排名，共45个学科进入全球TOP10，居全球第9位 。　　大家无语的是：今年在全美大学排54~也是没sei了。　　UW，是享有世界顶尖学术地位的著名研究型大学；1974年以来，它一直是全美联邦科研经费最有实力的竞争者之一，科研经费长期位居全球大学TOP3。　　建校156年来，它及校友创造出众多造福全人类的重大发明：发明乙肝疫苗、肾透析术；绘制人类基因图谱；主持设计世界上最大的波音747客机、月球轨道飞船和哥伦比亚航天飞机，培养了11位太空宇航员；开发计算机DOS操作系统……　　有13位诺贝尔奖、1位菲尔茨奖和2位图灵奖得主，167位美国科学委员会学部委员，252位美国院士，简直人才济济。　　　　　　2018　　2017　　这位选手降了1名，排名21。　　其实，它的历史最高综排达到全美第9位，在近20年的美国大学排行榜中，稳居TOP20。　　它的商学院的本科课程在著名杂志BusinessWeek里排美国第3，罗琳斯公共卫生学院全国TOP10，商学院、医学院和法学院的研究生院都在全美TOP20；此外，它在几乎所有其开设的学科领域，都位列全美TOP30。　　这也是一所全美顶尖的私立研究型大学，美国25所”新常春藤“院校之一，全美十大老牌名校之一，有“南部哈佛”的美誉，也是美国小班制精英教育的代表院校。　　可是，也有人提出：它的排名居然还在CMU、UVA、伯克利前面？这不科学！　　那么，这三大“劲敌”又是什么样的牛校呢？　　　　2018　　2017　　它今年排名25，比去年降了1名。　　它坐落在宾州的匹兹堡，其计算机学院和戏剧学院享誉全球，艺术学院、商学院、工程院以及公共管理学院等也在全美名列前茅。　　　　2018　　2017　　它与楼上一样，排名25，比去年降了1名。　　自USNews颁布全美大学排名以来，它始终名列全美最佳公立大学前列，是世界一流的公立研究型大学，最初的8所“公立常春藤”成员之一，25所“新常春藤”成员之一，也是北美唯一被联合国教科文组织列为是世界遗产的高等院校。　　曾被美国财经杂志《Kiplinger》和《普林斯顿评论》等权威评价机构连续多年评为“美国性价比最高的大学”。　　其知名校友中，包含两位美国总统及众多知名跨国集团公司创始人或CEO。　　　　2018　　2017　　嗯哪，加州体系来了。　　伯克利，今年排名21。　　它是世界著名的公立研究型大学，位列2016年ARWU世界大学学术排名世界第3、USnews世界大学排名第4。　　　　　　2018　　2017　　这位选手上升了2名。　　南加大，位于“天使之城”，是西海岸最古老的顶尖私立研究型大学，世界著名高等学府，是美国大学协会成员、环太平洋大学联盟成员。　　有资料显示，它2016年的本科录取率仅为16.5%，是全美本科最具竞争力且最难申请的大学之一。　　它的音乐学院、传播学院、公共政策学院、医药学院、建筑学院和多媒体学院等都居全美前列。2017年QS世界大学传播与媒体学专排第1位。　　它的电影艺术学院堪称顶尖，校友获奥斯卡奖数量居全美第一，斯皮尔伯格、“星战之父”乔治·卢卡斯等影坛巨匠是该校校友。 　　在历届奥林匹克运动会上，南加大校友累计获得的金牌数量和获奖总量也居全美高校第一。　　　　　　2018　　2017　　这位选手上升了3名。　　它常年稳坐泰晤士报全球大学排行榜TOP15，并在2017年US News全球大学排名中位列第10。　　UCLA，是加州大学系统中第三古老的分校，近年来在公立大学排名中高居第二名、仅次于加州伯克利。　　它也是申请人数最多的大学，是全美高中生梦寐以求的名校之一。　　它是美国商业金融、高科技产业、电影艺术等专业人才的摇篮，提供337个不同学科的学位，是全美培养尖端人才领域最广的大学之一；先后有16位教授和毕业生获诺贝尔奖。　　同时，它的学生或校友参加了自1928年以来每届奥运会，获得的总奖牌数位列全美第三。　　　　　　2018　　2017　　这位选手上升了2名。　　它是顶尖的公立大学，隶属于美国全国性第一级（Tier1）的大学；虽然成立才50余年，却位列加州大学系统内前三名，与UCLA在学术上不分伯仲。　　它也是“公立常春藤”中的顶尖者，曾产生20位诺贝尔奖得主，2位菲尔茨奖得主，是全美的学术发展重地；因此，大学研究拨款总额上，UCSD高达19亿美元，位居全美第5。　　另外，其环境优美气候宜人，坐拥全美顶级海滩，被Newsweek评为全美“最性感”的理科学习场所。　　　　　　2018　　2017　　这位选手上升了4名。　　它的主校区位于美国最富有州之一的康涅狄格州的曼斯菲尔德镇。该校2016财年的总经费预算高达28亿美元；是20所顶尖的公立大学之一，被誉为“公立常春藤”；是受新英格兰院校协会承认的高等院校，也是美国大东部竞技联盟的成员之一。　　它在教学和科研方面享有国际声誉：精算学专业全美知名，教育学院和商学院分别排31和48，其中，商学院以极高的性价比被《金融时报》评为全球百强商学院；教育专业第21。　　此外，杰出校友包括，诺贝尔物理学奖得主戴维·李、普利策奖得主Tim Page、前任天堂株式会社副主席George Harrison、前沃尔玛主席兼首席执行官William S。 Simon、NBA球星雷·阿伦等。　　　　　　2018　　2017　　这所学校比去年进步了7名，可有人还是感慨：捉急。　　它在2011年USNEWS的评选中超过哈佛，成为“最受美国学生和家长欢迎的美国大学”第1名。　　BYU，是著名的私立研究型大学，也是美国最大的教会大学，是美国第三大私立大学。　　其中，最佳学科是计算机科学、初级教育、工程、商科。其会计专业，曾被Business Week评为全美第1名，人力资源管理专业第27，美国研究生商科排名第27，法学院第33名。　　如今，它有在校生近30000人，其中，大部分是耶稣基督后期圣徒会的成员。大家被要求遵守荣誉规章，包括不饮酒、不吸毒、不抽烟，甚至不喝咖啡或茶，不能蓄胡，不能有婚前性行为等等。　　它设有多达4400个奖学会，此外，教会有专为教友开设的学生贷款，全校有六成学生都获得资助。　　　　　　2018　　2017　　这位选手两次排名很稳定。可有粉丝感叹：它可是从58名掉下来的！　　匹大是美国最早的十所大学之一，现已发展成世界闻名的顶尖大学，拥有极高学术声望，为“公立常春藤”院校之一，美国大学协会成员，也是美国东北地区最好的公立大学之一。　　它有每年研究经费六亿四千两百万美元的强劲研究实力，其科学及工程研究发展经费名列全美前20名，在整个联邦科学工程研究经费上占全美第13名。　　它下属的UPMC是全球顶尖医学院之一，在器官移植、神经外科学等领域领先国际；据2016 USNews报告，匹兹堡大学医学中心在全美医院机构中排名第5。　　　　　　2018　　2017　　这位选手上升了2名。　　它是世界著名的公立研究型大学，为明大系统历史最悠久，规模最大的分校；是美国十大联盟、美国大学协会的成员校之一，也是国际21世纪学术联盟成员之一，被誉为“公立常春藤”。　　化工系、机械工程系、心理系、经济学系、信息管理系、药学、森林、大众保健、教育系、 建筑系、景观系等学科在全美始终名列TOP10。　　飞行记录器、可收取式汽车安全带、心脏起搏器、心肺呼吸器等世界顶尖的技术和发明在明大诞生。　　建校至今，它的校友中，目前有26位诺贝尔奖得主，1位前美国首席大法官，2位前美国副总统，多位名列美国财富TOP500的企业领袖。　　　　　　2018　　2017　　IUB下降了4名。　　这是美国中北部的一所老牌名校，早在1909年便加入美国大学协会，也是美国30所“公立常春藤”之一；并与芝加哥大学、西北大学、威斯康星大学、密歇根大学等一同入选美国中西部十大联盟高校。　　它的Kelley商学院、Jacob音乐学院、环境与公共事务学院、信息与计算机学院在全美乃至世界享有盛誉。　　其中，Kelley商学院，自1920年创立以来，一直被认为是美国顶尖的商学院之一；Jacob音乐学院与Juilliard音乐学院、Eastman音乐学院并列全美第一；在公共事务学院中和哈佛的肯尼迪政府学院曾共同排名第2（所有公立大学中第一）。　　关于新出的排名，你还有什么槽点不吐不快呢？
